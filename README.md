## üì¶ Installation & Setup

1. **Clone the repo:**
   ```bash
   git clone https://github.com/heyGilmore/todo-list.git
   cd todo-list
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the app locally:**
   ```bash
   npx webpack serve
   ```

# Open http://localhost:8080/

4. **Build for production:**
   ```bash
   npx run build
   ```
